Case Overview

Auditors use Continuous Audit methodologies to systematically identify control failures within large-scale general ledger datasets. You will be building an automated scoring engine designed to flag potential procurement violations.

The Structuring Problem

"Structuring" occurs when an employee circumvents approval limits by splitting a large invoice into multiple smaller vouchers on the same day. You will build logic to catch this exact behavior.

The Branch Architecture

You will build a single, perfectly cleansed 'Master Dataset', and then branch off multiple 'Branches' to perform specific audit tests. Finally, you will rejoin these flags to build a single risk score per transaction.
